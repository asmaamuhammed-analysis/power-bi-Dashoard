# power-bi-Dashoard
I developed a comprehensive Project in power bi, creating multiple dashboards the tables to analyze the data. the process involved several stages. Including data processing, data cleaning, and data visualization

Analyze the global footwear Brands and create a report for it.
##Questions:
Compute the Total Sales of Product Category
Compare of the Famous Brands of the Footwear
Get the consumption Quantity of Each Country

Process:
1-	Download the Dataset
2-	Clean the Data By using Power Query
3-	Making some visuals by using FIGMA
4-	Build the visualization of the Dashboard By Power bi
